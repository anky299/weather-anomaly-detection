# Weather Anomaly Detection Tool 🌦️

This project is a simple web-based tool that analyzes weather data and tries to identify unusual patterns (anomalies) in temperature and rainfall.

The idea behind this project is to compare current weather conditions with historical averages and highlight when something looks “off” — for example, when the temperature is much higher or lower than expected.

---

## 🚀 What it does

* Fetches real-time weather data using OpenWeather API
* Compares it with historical data
* Detects temperature and rainfall anomalies
* Displays results in a clean dashboard with graphs
* Shows insights like:

  * Current vs average temperature
  * Rainfall comparison
  * Confidence level of anomaly detection

---

## 🧠 Why I built this

Weather patterns are becoming less predictable, and I wanted to build something that could visually show when current conditions differ from normal trends.

This project helped me understand:

* How to work with APIs
* Data comparison and basic analysis
* Building dashboards using Flask

---

## 🛠️ Tech Used

* Python (Flask)
* HTML, CSS, JavaScript
* OpenWeather API

---

## ▶️ How to run this project

1. Clone or download the repository

2. Open the project folder in terminal

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
